# Predicting-Term-Deposit-Suscriptions
Bank Marketing Campaign

## **Overview**
This project focused on analyzing a dataset related to a bank's term deposit marketing campaign. The main objective was to build a predictive model that can forecast whether customers will subscribe to the term deposit product or not, based on the customers' characteristics.

## **Approach**
**Data Collection and Preprocessing:** Loaded the data, checked and handled missing values, removed duplicates.
**Data Exploration:** Performed statistical analysis and exploration of the data features, including distribution, correlations, etc.
**Data Preprocessing:** Encoded categorical features, normalized numeric features.
**Model Building:** Selected appropriate machine learning techniques, trained and evaluated the model performance.

## **Results**
After processing and analysis, a classification model was developed to predict whether customers will subscribe to the term deposit product. The model achieved high accuracy, with an AUC-ROC metric around 0.8, indicating strong predictive capability.
The analysis also identified the key factors influencing the customer's decision, providing valuable insights for the bank's marketing department.

## **Conclusion**
This project successfully built an effective predictive model that can help the bank improve its term deposit marketing campaign and enhance customer targeting and retention strategies. The data analysis findings also provided valuable information to support the bank's marketing activities.
